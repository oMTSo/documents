一阶电路 (First-Order Circuits)
#### RC 电路
根据基尔霍夫电压定律 (KVL)，回路方程为：

$$Ri(t) + v_C(t) = u_s(t)$$

将 $i(t) = C \frac{dv_C}{dt}$ 代入，得：

$$RC \frac{dv_C(t)}{dt} + v_C(t) = u_s(t)$$

#### RL 电路
根据 KVL：

$$L \frac{di(t)}{dt} + Ri(t) = u_s(t)$$
- **时间常数:** $\tau = \frac{L}{R}$

二阶电路 (Second-Order Circuits)
当电路中同时包含 $L$ 和 $C$ 时（如 RLC 串联电路），会产生二阶微分方程。
#### RLC 串联电路
根据 KVL：

$$L \frac{di(t)}{dt} + Ri(t) + \frac{1}{C} \int_{-\infty}^t i(\tau) d\tau = u_s(t)$$

对时间 $t$ 求导，得到关于电流 $i$ 的标准二阶微分方程：

$$L \frac{d^2i(t)}{dt^2} + R \frac{di(t)}{dt} + \frac{1}{C} i(t) = \frac{du_s(t)}{dt}$$

若以电容电压 $v_C(t)$ 为变量，则方程为：
$$LC \frac{d^2v_C(t)}{dt^2} + RC \frac{dv_C(t)}{dt} + v_C(t) = u_s(t)$$

# 常用微分方程公式
- 电容 (Capacitor): 电流与电压的变化率成正比
$$i(t) = C \frac{dv(t)}{dt}$$
- 电感 (Inductor): 电压与电流的变化率成正比
$$v(t) = L \frac{di(t)}{dt}$$
