# 降阶与状态空间 (State-Space)

在复杂的工程仿真（如你用的 MATLAB 或 Adams）中，二阶系统通常被看作是两个相互耦合的一阶系统。

例如，位移的导数是速度，速度的导数是加速度。通过引入“状态变量”，一个二阶微分方程可以转化为两个一阶微分方程组。这意味着二阶系统在逻辑上是一阶系统的“进化版”。

## 包含关系

当二阶系统的阻尼比 $\zeta$ 非常大（过阻尼）且其中一个极点远大于另一个时，它的行为会非常接近一阶系统。在工程简化中，我们经常把这种“迟钝”的二阶系统简化为一阶系统来处理。

## 能量的演变

- **一阶系统**只有一种储能方式，能量只能被“充满”或“放光”。
    
- **二阶系统**有两种储能方式，能量不仅可以充放，还可以在两种形式之间**跳舞（振荡）**。

# 以 RLC串联电路为例
关于电流 $i$ 的标准二阶微分方程：

$$L \frac{d^2i(t)}{dt^2} + R \frac{di(t)}{dt} + \frac{1}{C} i(t) = \frac{du_s(t)}{dt}$$

若以电容电压 $v_C(t)$ 为变量，则方程为：

$$LC \frac{d^2v_C(t)}{dt^2} + RC \frac{dv_C(t)}{dt} + v_C(t) = u_s(t)$$
对于 RLC 串联电路，选取 $v_C$ 和 $i_L$ 为状态变量：

$$\begin{cases} \frac{dv_C}{dt} = \frac{1}{C} i_L \\ \frac{di_L}{dt} = \frac{1}{L} (u_s - Ri_L - v_C) \end{cases}$$
