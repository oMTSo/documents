# 传递函数
首先，从[[传递函数]]入手

传递函数=系统输出的拉氏变换/系统输入的拉氏变换
同时
传递函数=冲激响应的拉氏变换

因此，传递函数实际的物理意义是：给一个系统瞬间充能([[零输入响应]])(狄拉克函数包含了全频域)后，系统对于全频域上的能量的增益(==这里的增益，理解为响应能力或匹配度会好一点，本质上就是系统在多大程度上愿意让这种形状的能量通过或放大==)。

## 特征方程
![image.png|350](https://raw.githubusercontent.com/oMTSo/images/main/img/20260108232155701.png)

传递函数（Transfer Function）的**特征方程**（Characteristic Equation）是自动控制理论中的一个核心概念，它决定了系统的“本能”——也就是你之前提到的**系统固有属性**。

简单来说，如果传递函数是一个分式：

$$G(s) = \frac{N(s)}{D(s)} = \frac{\text{分子多项式}}{\text{分母多项式}}$$

那么，令分母多项式等于零所构成的方程，就叫做特征方程：

$$D(s) = 0$$
==这里主要讨论闭环特征方程，也就是：1+开环传递函数==

## 极点
极点是分母方程 $D(s) = 0$ 的根。从原理上讲，**极点决定了系统自然响应（即冲激响应）的函数形式。**

### 数理原理：部分分式展开

当你对一个传递函数进行逆变换时，数学上会进行部分分式展开：

$$G(s) = \frac{A}{s - p_1} + \frac{B}{s - p_2} + \dots$$

根据逆变换公式 $\mathcal{L}^{-1} \{ \frac{1}{s-p} \} = e^{pt}$，每一个极点 $p$ 都会在时域产生一个对应的项 $e^{pt}$。

### 性能如何体现？

极点 $p = \sigma + j\omega$ 在复平面上的位置直接告诉了我们：

- **稳定性（实部 $\sigma$）**：
    
    - 如果 $\sigma < 0$，项 $e^{-|\sigma|t}$ 会随时间衰减。**实部离虚轴越远（越负），系统平稳得越快（调节时间短）。**
        
    - 如果 $\sigma > 0$，系统爆炸。
        
- **振荡特征（虚部 $\omega$）**：
    
    - 如果存在虚部 $j\omega$，时域就会出现 $\sin(\omega t)$。**虚部越大，系统晃动得越快（频率高）。**
        
- **阻尼比（极点与原点的连线夹角）**：
    
    - 极点靠近实轴，阻尼大，不爱晃；极点靠近虚轴，阻尼小，晃个不停。
        

> **总结**：极点定义了系统的“自然模态”。它告诉我们：当外界干扰撤去后，系统内部残余能量将以何种频率和衰减率（$e^{st}$）释放。极点也是系统的“共振点”，当输入信号的形状与极点匹配时，系统会表现出最大的响应灵敏度。

## 零点
零点是分子方程 $N(s) = 0$ 的根。从原理上讲，**零点并不改变响应的函数形式（还是那几个 $e^{pt}$），但它改变了每一项的系数（权重）。**

### 数理原理：残数（Residues）的影响

在部分分式展开中，每一项前面的系数 $A, B$ 是由零点的位置决定的。

- 如果一个零点靠近某个极点，它会显著**削弱**该极点对应的响应分量。
    
- 如果零点正好重合在极点上（相消），该极点的响应会彻底消失。
    

### 性能如何体现？

- **减小超调或增加超调**：零点可以看作是对输入信号的“微分”处理。左半平面的零点通常会使响应变快，但也可能导致过大的超调。
    
- **反向过冲（右半平面零点）**：如果零点在右半平面（非最小相位系统），系统在起跳瞬间会先往反方向走。就像你猛打方向盘左转，车屁股先往右摆一下。
    
- **滤波特性**：在频域，零点是“陷波器”。如果某个频率 $s$ 靠近零点，该频率的信号就会被大幅度衰减。
    

> **总结**：零点是信号能量全部被吸收消耗的模态。任何这样的信号都会在系统内部被消耗。

## 理解G(s)值的大小
当G(s)的值为正无穷时，说明当前模态是系统的自然模态，任何属于这个模态的信号都不会被额外消耗，而是被系统保留，并以这个模态的衰减率自然消耗
当G(s)的值为零时，当前模态会被系统全部吸收
当G(s)的值为1时，输入=输出，是控制的最佳状态
